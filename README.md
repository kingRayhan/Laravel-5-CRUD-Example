# laravel-cruddy example.
This is a preconfigured example for [laravel-cruddy](https://github.com/Askedio/laravel-Cruddy), a CRUD JSON API for Laravel 5.2.


# Install
* clone this repo

~~~
composer install
~~~

* configure .env with a database

~~~
composer migrate
php artisan serve
~~~
